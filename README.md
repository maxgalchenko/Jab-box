## Overview

Responsive, static multipage gym website demonstrating layout, mobile navigation, and scroll-triggered animations with GSAP. Useful as a simple reference for HTML/CSS structure and basic interactive behaviors without a build system.

## Key Features

- Responsive layout with mobile hamburger menu
- GSAP-powered entrance and scroll-triggered animations
- Multipage structure: Home, Services, Plans, Trainers

## Tech Stack

HTML5, CSS3, JavaScript (ES6), GSAP 3, Font Awesome 5

## Architecture

Static multipage site (`index.html`, `services.html`, `plans.html`, `trainers.html`) with client-side JavaScript for menu control and animations. No build tools, backend, or package manager.

## Performance & Accessibility

Lightweight, framework-free pages with semantic HTML and responsive CSS. Images are served statically; animations are optimized via GSAP timelines and ScrollTrigger.

## Prerequisites

- None (static site)

## Installation

```bash

cd Jab-box
```
